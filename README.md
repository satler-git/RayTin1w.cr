## Installation

Clone the repo.

```bash
git clone https://github.com/satler-git/RayTin1w.cr.git
```

And install shards.

```bash
cd RayTin1w.cr
shards install
```

## Usage

Write to ppm.

```bash
crystal run ./src/raytin1w.cr > image.ppm
```
